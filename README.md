## Getting Started

### Clone Repository

```bash
git clone https://github.com/Vibhupatel7/portfolio.git
cd portfolio
```

### Install Dependencies

Using Bun:

```bash
bun install
```

Using npm:

```bash
npm install
```

### Start Development Server

```bash
bun run dev
```

or

```bash
npm run dev
```

Open:

```bash
http://localhost:5173
```
